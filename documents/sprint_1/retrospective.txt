Pour le sprint suivant nous avons décidé d'accélérer le développement afin de réaliser plus de fonctionnalités. Ceci sera permis grâce à l'indépendances des tâches à effectuer dans ce sprint.
L'objectif est tout d'abord d'avoir une véritable expérience de jeu.
- Dans un premier temps, nous souhaitons avoir une génération pseudo-aléatoire des niveaux
  (on partirait d'une base de labyrinthe défini dans un fichier, avec quelques modifications aléatoires des cases)

- Dans un deuxième temps les monstres ajouteront de la difficulté au joueur

- Le jeu devra également avoir un système d'armes simples pour se défendre face aux monstres.

- Des cases spéciales vont permettre d'ajouter de l'interaction au jeu

- Le jeu devra être doté d'une interface graphique avec un menu de jeu

Le projet a bien avancé :
  - Les fonctionnalités prévues pour le sprint 1 ont été réalisées (certaines différemment que prévu initialement)
  - Tous les membres de l'équipe ont travaillés équitablement (sauf pour les diagrammes, voir partie cependant)
  - Il reste du travail à faire pour les itérations suivantes

Cependant :
  - Quelques difficultés pour réaliser les diagrammes, les fonctionnalités et leurs tests puisque 3 personnes du groupe ont été malades durant ce sprint.
  - Pour cette raison le travail s'est surtout concentré sur la fin du sprint.


Pour le sprint suivant nous avons décidé de mieux adapter le développement au temps que nous auront afin de réaliser un code de meilleure qualité tout en travaillant sur les autres projets. 

L'objectif est tout d'abord d'avoir une meilleure expérience de jeu (faire en sorte que toute les fonctionnalités du jeu soient implémentées.
- Dans un premier temps, nous souhaitons avoir une interface graphique amélioré (plus conviviale et avec de nouveaux sprites)

- Dans un deuxième temps, les bugs résiduels devront être corrigés et le code amélioré. 
  Il va falloir faire une refonte de certaines méthodes et classes pour optimiser les calculs et éviter la duplication de code.

- Les cases spéciales devront être mieux gérées et de nouvelles seront ajoutées.

- Le système d'armes va devoir être amélioré.
